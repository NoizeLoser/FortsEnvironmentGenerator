# Forts Environment Mod Generator 🛠️🎬

Современный графический генератор структуры модов окружения (карт/тем оформления) для игры **Forts**. Написан на Python с использованием библиотеки `CustomTkinter` и поддерживает кастомные темы интерфейса.

## ✨ Основные возможности
* **Управление слоями (Параллакс):** Удобная таблица для настройки `ZoomFactor`, скоростей смещения по X/Y и выбора файлов текстур через проводник.
* **Разделение Бэк/Фронт:** Возможность выносить погодные слои (снег, дождь, туман) на передний план поверх боевых фортов (`Foreground = true`).
* **Аудио-менеджер:** Быстрая привязка MP3/OGG треков для фонового эмбиента карты и 4 игровых состояний музыки (Idle, Intense, Win, Lose) с автоматическим копированием в структуру мода.
* **Видео-предпросмотр:** Автоматическая генерация цикличной GIF-анимации для визуальной проверки настроек параллакса прямо перед сборкой мода.

## 💻 Инструкция для разработчиков (Запуск из исходного кода)

Если вы хотите изменить код или запустить его вручную, вам понадобятся файлы `generator_ui.py` и `generator_backend.py`.

### 1. Установка зависимостей
Убедитесь, что у вас установлен Python 3.8+, и выполните команду в терминале:
```bash
pip install customtkinter Pillow darkdetect
```

### 2. Запуск приложения
```bash
python generator_ui.py
```

### 3. Компиляция собственного `.exe`
Для сборки проекта в один исполняемый файл используется `pyinstaller`:
```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --add-data "red_theme.json;." --add-data "FortsEnvironmentGeneratorIcon.ico;." --icon="FortsEnvironmentGeneratorIcon.ico" generator_ui.py
```
После сборки готовый файл появится в папке `dist/`.
